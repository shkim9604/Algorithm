# Lv1 약수의합
 [약수의 합 링크]("https://school.programmers.co.kr/learn/courses/30/lessons/12928")

### 테스트결과
 Time:0.15ms, Memory: 9MB
### 문제설명
<p>정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.</p>

<h5>제한 사항</h5>

<ul>
<li><code>n</code>은 0 이상 3000이하인 정수입니다.</li>
</ul>

<h5>입출력 예</h5>
<table class="table">
        <thead><tr>
<th>n</th>
<th>return</th>
</tr>
</thead>
        <tbody><tr>
<td>12</td>
<td>28</td>
</tr>
<tr>
<td>5</td>
<td>6</td>
</tr>
</tbody>
      </table>
<h6>입출력 예 설명</h6>

<p>입출력 예 #1<br>
12의 약수는 1, 2, 3, 4, 6, 12입니다. 이를 모두 더하면 28입니다.</p>

<p>입출력 예 #2<br>
5의 약수는 1, 5입니다. 이를 모두 더하면 6입니다.</p>

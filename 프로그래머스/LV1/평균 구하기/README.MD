### 문제설명
<p>정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.</p>

<ul>

<li>arr의 원소는  -10,000 이상 10,000 이하인 정수입니다.</li>
</ul>
